# Library Management System - Java

A Java library management system built using object-oriented programming.  
The project manages books, users, borrowing records, and basic library operations.

## Features

- Add and manage books
- Add and manage users
- Borrow books
- Return books
- Track borrowing records
- Organize library operations using separate Java classes

## Technologies Used

- Java
- Object-Oriented Programming
- Classes and Objects
- Array/List-based data handling
- Basic system design

## Project Files

- `App.java` - Main program entry point
- `Book.java` - Represents book information
- `User.java` - Represents library users
- `BorrowingRecord.java` - Tracks borrowing and returning records
- `Library.java` - Handles main library operations

## How to Run

Compile the Java files:

javac *.java
